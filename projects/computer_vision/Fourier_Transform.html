Overview

A problem set to explore the Fourier Transform. Implemented the discrete Fourier Transform then used it to compress an image by filtering out the least significant frequencies. Finally, used the Forier Transform to create a low-pass filter and demonstrate the difference in compressed image quality and illustrate the difference in the frequency domain.

Technical Details

My solution was written in Python using openCV and numpy for basic image manipulation and calculations. No use of built in fft algorithms allowed!

Key Features

  • Image Compression: Use of Discrete Fourier Transform for image compression at different thresholds.
  • Low-pass Filter: Use of transform to low-pass filter an image at different thresholds
